sort_zr_intermediate_ascii: 
Sort the radar and gauge merged ascii file.
Updated: February 9, 1999.

Synopsis

    Usage: sort_zr_intermediate_ascii [second_intermediate.ascii]

Description

Sort the radar and gauge merged file, aka the 2nd intermediate file by: net, gauge#, date and time.

Options

None.

Input

STDIN or the merged radar and gauge file generated by merge_radarNgauge_data.

Output

A table of R and G accumulations suitable to gauge_gui_main.pl or r_over_g.pl.

See also

automatic_zr.pl, gauge_radar_accum.pl, gauge_gui_main.pl.

Author
