r_over_g.pl: 
Given the merged radar and gauge ascii file, generate a table of R/G for each gauge, net and site for Convective and Stratiform.
Updated: February 9, 1999.

Synopsis


    gauge_radar_accum.pl [s.ascii] | r_over_g.pl [--help] [--total-only]
 

Description

Take the output of gauge_radar_accum.pl and generate a summary
 table that shows the R/G for each gauge, the net and the entire
 file for each rain type: stratiform, convective, etc.

  The input does not need to be sorted.

 Example usage:

 gauge_radar_accum.pl radar_merged.ascii | r_over_g.pl

Options

    --total-only   - Generate R and G sums, don't perform R/G.
    --help         - Print the usage.

Input

The output of the command gauge_radar_accum.pl.

Output

Table of R/G for each gauge, network, site and the total R/G for the entire input file.  Here is an example, the command is:  gauge_radar_accum.pl s.ascii | head | r_over_g.pl.  Each gauge is listed by network and gauge number and the stratiform and convective r/g for them are presented.  At the end of the gauge list, you'll see just the network entries.  The R/G for these represent the total Rtotal/Gtotal for the entire network.  The Total entry shows the R/G for the entire input file -- it could be interpreted as the site R/G.
