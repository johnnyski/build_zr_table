gauge_radar_accum.pl: 
Generate radar and gauge rain accumulations for each raintype.
Updated: February 9, 1999.

Synopsis

    Usage: gauge_radar_accum.pl [--correlation-index n]
                         [--negative]
                         [--Network]
                         [--tmean x]
                         [--Rainrate]
                         [--help]
                         [radar_gauge_merge.ascii]
 
 

Description

Generate a table of R and G rain accumulations for each gauge for each rain type.  Currently there are only two raintypes: Convective and Stratiform.  The input file should be sorted with the program sort_zr_intermediate_ascii.
The output of this program is useful to gauge_gui_main.pl for graphical display of the rain accumulations over time and to the program r_over_g.pl which generates a comprehensive R/G table for each gauge and for each raintype.

Options

   --matrix-index n  - Specify the matrix index (n) to plot.  The matrix
                       contains the radar to gauge correlations, it is a 3x3
                       matrix with the center being over the gauge.
                       Usually, 0 through 7 for the 3 by 3 matrix.
                       Default = 4 (this is the center).
   --negative   - Accumulate negative gauge values.  Negative
                  numbers are suspect, however, they're good for
                  rain totals.  In other words, INCLUDE negative gauge values.
                  Default: Use positive gauge values only.
   --Network    - Accumulate for entire network.
                  Default: accumulate for each gauge.
   --tmean n    - Number of gauge measurements to average.  Default: All.
                  'n' is typically [5..15] and odd.
   --Rainrate   - Data is already in mm/hr.  Default: treat data as dBZ.
   --help       - Print this usage.

Input

STDIN or the merged radar and gauge file generated by merge_radarNgauge_data.

Output

A table of R and G accumulations suitable to gauge_gui_main.pl or r_over_g.pl.

See also

automatic_zr.pl, gauge_radar_accum.pl, gauge_gui_main.pl.
